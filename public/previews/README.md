# Videos e imágenes para la página Landing

La página **Landing** (/landing) está diseñada como oferta de venta de sitios web. En la sección **"Así se ven los sitios que creamos"** se muestran **videos** de cada sitio.

## Videos de preview (cómo se ve cada sitio)

Subí un video corto mostrando cómo se ve el sitio en el navegador (scroll, clicks, etc.) y guardalo en esta carpeta con estos nombres:

| Sitio                    | Nombre del archivo              |
|--------------------------|---------------------------------|
| Rogerbox                 | `rogerbox-web.mp4`              |
| Martha Lucia Contreras   | `marthaluciacontreras-web.mp4`  |
| Fundesae                 | `fundesae-web.mp4`              |

- Formato: **MP4** (o el que prefieras; si cambias la extensión, actualizá la ruta en `src/app/landing/page.tsx`).
- Duración: 15–45 segundos está bien (se reproduce en loop, muteado).
- Contenido: pantalla grabada del sitio en vivo (desktop o móvil).

C&M se muestra como "Próximamente" hasta que tengas el sitio y el video.
